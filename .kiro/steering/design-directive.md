# Frontend Design Directive — AndyTipster V2

## Design Philosophy
- Premium SaaS platform for professional sports analysis
- Rich Cricket Green primary colour with gold/amber accents
- Left sidebar navigation as primary nav
- Information dense without feeling crowded
- Calm, modern, organised, premium appearance

## Colour Palette (from reference image)
- Primary (Forest): #0B5E1C
- Dark Green: #064E28
- Deep Green: #043D22
- Accent (Mint): #23C55E
- Warning (Amber): #F1BF24
- Danger: #EF4444
- Light Grey bg: #F4F6F5
- Cards: #FFFFFF
- Sidebar: Dark green gradient
- Slate text: #192938

## Layout Rules
- LEFT SIDEBAR first (primary navigation)
- Top bar: logo, search, notifications, profile
- Pages: Title → Description → Actions → Filters → Content
- Use tabs, tables, accordions — avoid card-inside-card
- Tables are first-class citizens for business data
- Mobile: sidebar becomes drawer

## Typography
- Font: Poppins
- H1: 32px / Bold
- H2: 24px / SemiBold
- H3: 18px / SemiBold
- Body: 14px / Regular
- Small: 12px / Regular

## Do NOT
- Use pure white backgrounds (use #F4F6F5)
- Use neon greens, harsh blacks, bright reds
- Create casino/gambling aesthetic
- Overuse cards — tables and lists preferred
- Use flashy animations
